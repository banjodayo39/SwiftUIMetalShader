# Metal Shader SwiftUI 

Example Project of how to use Metal Shader with SwiftUI 

## Features
- Layer Effect
- Distortion Effect
- Color Effect

## Requirements
- Xcode (version 15.0 beta or later)
- iOS Deployment 17.0
- Swift 5.9

## Installation
- Clone or download the project repository.
